Text to 3D Braille Generator

Welcome to the Text to 3D print web app. In this page, you can translate your text to 3D brailleand preview the file prior to sending it to us to be printed. To use the tool you must:

  • Choose what language you are translating from, at the moment we only support Danish and English
  • Write a short description about how this sign will be used

Please provide a description of where this sign will be and what purpose it will serve:

If you are happy with the generated 3D model, you can send it to us so that we print it. Our current wait time for prints is around 1 week
